# Scenario 6: Manage and Observe

## Description
After deployment, users can manage environments and observe their components via logs, metrics, and traces. Same capabilities as Integration Platform.

## User Scenarios

### 1. View Runtime Logs
- User navigates to a deployed component → Logs tab
- User sees runtime logs from OpenChoreo Observer API

### 2. View Metrics
- User views component metrics (request rate, latency, error rate)

### 3. View Traces
- User views distributed traces for debugging

### 4. Manage Environments
- User views and manages environment configuration
- User can start/stop deployments per environment
